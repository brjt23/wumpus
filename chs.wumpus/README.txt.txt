***********************************************

-------------------------------------------------------
Requirements: 
-Java 1.8+
-Maven apache-maven-3.6.0
-------------------------------------------------------

To run the application:
-Download the sources 
-Go to ../wumpus/chs.wumpus/. 
-Open a console and run the command 
“mvn clean install”
-Then go to ../wumpus/chs.wumpus/target/ there you will find the Wumpus-0.0.1-SNAPSHOT.jar and run
“java -jar Wumpus-0.0.1-SNAPSHOT.jar”
The application will start.

To get more info about how the game works you can read INSTRUCTIONS.txt
